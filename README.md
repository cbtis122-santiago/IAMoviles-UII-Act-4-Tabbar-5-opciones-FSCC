Hola, soy Santiago Carrasco. Presento mi proyecto de aplicación móvil para mi negocio 'AlaBurger'. Es una app desarrollada en Flutter con un diseño funcional que incluye una barra de navegación inferior (BottomNavigationBar) con 5 secciones: Inicio, Menú, Pedidos, Ubicación y Perfil. El código está organizado de forma limpia usando rutas nombradas y una estructura de carpetas modular (mis_paginas_tab). Además, todas las imágenes se cargan dinámicamente desde mi repositorio de GitHub.
